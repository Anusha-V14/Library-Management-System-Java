# Library Management System (Java)

## Description
A desktop-based Library Management System built using Java Swing, JDBC, MySQL, and Maven.

## Features
- Admin Login
- Add Books
- Issue Books
- Return Books
- MySQL Database Integration

## Technologies Used
- Java (Swing)
- JDBC
- MySQL
- Maven

## How to Run
1. Import project into STS/Eclipse
2. Configure MySQL database
3. Run Login.java
4. Login with:
   - Username: admin
   - Password: admin123
